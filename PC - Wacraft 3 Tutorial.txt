Hero XP Gained - Hero, Table = 100,120,160,220,300,500

Experiência ganha ao matar um Herói,sendo ele : Lv 1 = 100 xp
Lv 2 = 120 xp
...
Lv 6 = 500 xp

Experiência ganha = Hero XP Gain * Hero Previous value * Previous valot factor + Level *  Level Factor + Constant Factor

Exemplo : Matar um Herói Lv 7
Previous value ( Valor anterior citado noa Hero XP Gained - Hero, Table ) = 500
Hero XP Gained - Hero, Previous value factor = 1
Level do Herói : 7
Hero XP Gained - Hero, Level factor: 0
Hero XP Gained - Hero, Constant factor: 100

500*1+7*0+100
500 + 0 + 100
600 xp

Já um Herói level 8

600*1+8*0+100
600 + 0 + 100
700
